### 贡献组件

- clone 项目到本地
```sh
git clone git@git.bilibili.co:crayon/sakura.git
```

- 下载依赖（建议 yarn）
```sh
yarn
```

- 启动文档的 server
```sh
npm run dev
```
- 在`src/components`目录下新建一个你组件的目录（仿照其他组件即可）

- 在`demo/pages`目录下新建一个你组件的预览页面（仿照其他组件即可）

- 在`docs/components`目录下新建一个你组件的文档页面（仿照其他组件文档引入`Phone`预览组件）

- 开始开发
